การเปรียบเทียบ OOP ใน Java, C++, และ Python

ความเหมือน

  • หลักการ OOP: ทุกภาษามีการใช้หลักการ OOP เช่น Class, Object, Encapsulation, Inheritance, Polymorphism และ Abstraction
  • การใช้งาน Class และ Object: สร้างคลาสและออบเจกต์เพื่อจัดเก็บข้อมูลและพฤติกรรมได้
  • การสืบทอด: สามารถใช้ Inheritance เพื่อสืบทอดคุณสมบัติจากคลาสหนึ่งไปยังอีกคลาสหนึ่ง
  • การใช้ Constructors: มีการใช้ Constructors เพื่อกำหนดค่าเริ่มต้นสำหรับออบเจกต์
  • Polymorphism: การสร้างเมธอดที่มีชื่อเดียวกันแต่ทำงานต่างกันได้

ความแตกต่าง

คุณสมบัติ Java C++ Python
การจัดการหน่วยความจำ ใช้ Garbage Collection ผู้พัฒนาจัดการเองโดยใช้ new และ delete ใช้ Garbage Collection และ reference counting
การสืบทอดหลายคลาส (Multiple Inheritance) ไม่รองรับโดยตรง ใช้ Interfaces แทน รองรับการสืบทอดหลายคลาส รองรับการสืบทอดหลายคลาส
ความเข้มงวดในการพิมพ์ตัวแปร Static Typing ต้องระบุตัวแปรชัดเจน Static Typing ต้องระบุตัวแปรชัดเจน Dynamic Typing ไม่ต้องระบุตัวแปรชัดเจน
ความง่ายในการเขียน ต้องระบุชนิดตัวแปรชัดเจนและโครงสร้างเคร่งครัด ต้องระบุชนิดตัวแปรและโครงสร้างซับซ้อนกว่า ไวยากรณ์เรียบง่ายและกระชับกว่า
การจัดการข้อยกเว้น (Exception Handling) ใช้ try-catch-finally ใช้ try-catch ไม่บังคับให้จัดการข้อยกเว้น ใช้ try-except-finally ไม่บังคับการจัดการข้อยกเว้น
การทำงานร่วมกับโค้ดระดับล่าง ทำงานในระดับสูง ไม่เข้าถึงฮาร์ดแวร์โดยตรง สามารถทำงานกับโค้ดระดับล่างได้ดี ทำงานในระดับสูงเป็นหลัก

ตัวอย่างโค้ด

Java

class Animal {
    String name;

    Animal(String name) {
        this.name = name;
    }

    void sound() {
        System.out.println("Animal makes sound");
    }
}

class Dog extends Animal {
    Dog(String name) {
        super(name);
    }

    @Override
    void sound() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Dog("Buddy");
        dog.sound();
    }
}

C++

#include <iostream>
using namespace std;

class Animal {
public:
    string name;
    
    Animal(string name) : name(name) {}

    virtual void sound() {
        cout << "Animal makes sound" << endl;
    }
};

class Dog : public Animal {
public:
    Dog(string name) : Animal(name) {}

    void sound() override {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Animal* dog = new Dog("Buddy");
    dog->sound();
    delete dog;
    return 0;
}

Python

class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        print("Animal makes sound")

class Dog(Animal):
    def sound(self):
        print("Dog barks")

dog = Dog("Buddy")
dog.sound()

สรุป

Java มีโครงสร้างที่เข้มงวดและทำงานในระดับสูง โดยมีการจัดการหน่วยความจำอัตโนมัติด้วย Garbage Collection เหมาะสำหรับการพัฒนาโปรแกรมที่ต้องการความปลอดภัยและควบคุมข้อผิดพลาด

C++ มีความยืดหยุ่นสูงและสามารถทำงานในระดับต่ำได้ดี แต่ต้องจัดการหน่วยความจำด้วยตัวเอง ใช้ในการพัฒนาระบบที่ต้องการประสิทธิภาพสูง เช่น ระบบปฏิบัติการหรือซอฟต์แวร์ที่ต้องทำงานใกล้ชิดกับฮาร์ดแวร์

Python เป็นภาษาที่เขียนได้ง่ายและกระชับ เหมาะสำหรับการพัฒนาอย่างรวดเร็วและการทำงานในระดับสูง แต่มีประสิทธิภาพน้อยกว่าในงานที่ต้องการความเร็ว ใช้ในงานที่เน้นความสะดวก เช่น งานวิจัย การพัฒนาเว็บ หรือการประมวลผลข้อมูล
